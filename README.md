# Naman Scale Quotation Generator

A modern web-based quotation generator for Naman Scale weighing machines business in Bharatpur.

## Features

- **Easy Quotation Generation** - Fill customer details and scale specifications
- **Automatic Pricing** - Prices calculated based on scale type and quantity
- **GST Calculation** - Automatic 18% GST calculation for Indian market
- **Print Ready** - Generate print-friendly quotations
- **Professional Layout** - Clean and professional quotation format

## Scale Types & Pricing

- **Platform Scale** - ₹5,000
- **Mechanical Scale** - ₹3,000
- **Digital Scale** - ₹8,000
- **Hanging Scale** - ₹4,000

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```

## Deployment

This project is set up for deployment on Vercel with automatic deployments from GitHub.

## Technologies Used

- **Next.js 14** - React framework
- **React 18** - UI library
- **CSS-in-JS** - Inline styling
